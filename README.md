# Landing Page
A responsive, landing page is developed as part of a web development internship task. This project demonstrates skills in semantic HTML5 layout, CSS3 styling, and responsive design principles.
## Project Overview
* **Project Name:** Landing Page
*  **Type:** Internship Project (Level 1 Task)
*  **Organization:** Oasis Infobyte
*  **Status:** Completed

This landing page is designed for a fictional business service, featuring a clean UI with a hero section, feature highlights, and a contact footer. The design is fully compatibile across desktop and mobile devices.

## Key Features
Based on the project implementation:

* **Responsive Navigation Bar:** Includes a logo and navigation links that adapt to screen size, utilizing a hamburger menu structure for mobile views.
* **Hero Section:** Features a compelling headline ("Build Your Future With Us"), subtext, and a "Get Started" Call-to-Action (CTA) button with hover effects.
* **Features Grid:** A dedicated section displaying key service attributes ("Fast Performance", "Secure Data", "24/7 Support") using a flexible card layout.
* **Modern Styling:**
    * **Typography:** Uses the 'Poppins' font family from Google Fonts for a clean, modern look.
    * **Smooth Scrolling:** Implemented `scroll-behavior: smooth` for seamless navigation between sections.
    * **Interactive Elements:** CSS hover effects on buttons and cards (transformations and shadow changes) to enhance user experience.
* **Footer:** A professional footer section with copyright information ("2025 Intern Project").

##  Technologies Used

* **HTML5:** For the structural skeleton of the web page.
* **CSS3:** For styling, including:
    * **Flexbox:** Used extensively for aligning the navbar, hero content, and feature cards.
    * **Media Queries:** Implemented (`@media (max-width: 768px)`) to handle mobile responsiveness.
    * **CSS Variables:** Used for consistent color management.
* **VS Code:** Development environment.
